## 🚀 Como rodar localmente

### 🔧 Pré-requisitos

- Python instalado
- MkDocs com tema Material:
```bash
pip install mkdocs-material
```

### ▶️ Rodar localmente

```bash
mkdocs serve
```
Acesse:
```
http://127.0.0.1:8000
```

### 📦 Gerar o site estático

```bash
mkdocs build
```

### 🚀 Publicar no GitHub Pages

```bash
mkdocs gh-deploy
```
