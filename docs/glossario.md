# Glossário do Larjam

| Termo         | Definição                                    |
|----------------|----------------------------------------------|
| Nomi          | Unidade de informação, conceito ou padrão.   |
| Nomus         | Um conjunto de nomi alinhados.               |
| Instanciamento| Processo de ativar um nomi em um eu.         |
| Entranhamento | O enraizamento de um nomi na mente.          |
